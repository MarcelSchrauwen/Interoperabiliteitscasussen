BgGGZ ()
De BgGGZ is een gegevensdienst binnen MedMij en omvat in eerste aanleg 23 zorginformatiebouwstenen (zibs). Deze bouwstenen zijn deels ontleend aan de bouwstenen BgZ en zijn voor een gedeelte aangevuld met GGZ-specifieke bouwstenen. Deze gegevensdienst is een verplicht onderdeel in de VIPP GGZ-regeling. Mogelijk wordt de dienst de komende jaren verder uitgebreid. website VZVZ: https://www.aorta-lsp.nl/zorgtoepassingen/bgggz
