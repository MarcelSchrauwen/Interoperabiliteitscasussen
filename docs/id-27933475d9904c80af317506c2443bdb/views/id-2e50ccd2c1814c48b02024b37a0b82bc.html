02. Plateaus & kenmerken ()
OPMERKING van Marcel: Voor het best passende archimate element om de kenmerken van het gezondheidsinformatiestelsel weer tegeven kom ik uit op "capabiliy" oftewel vaardigheid of vermogen (niet financiëel). Oftewel: Welke capabilites moet het gezondheidsinformatiesysteem bevatten om goed te kunnen functioneren. Deze capabilities zijn later te vertalen naar organisatorische of technische functies.
