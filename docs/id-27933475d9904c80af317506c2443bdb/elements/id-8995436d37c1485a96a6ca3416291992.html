System Use Cases ()
De System Use Case beschrijft het gedrag (de functionaliteit) van een onderliggend systeem / een digitale voorziening. Het systeem kan daarbij een verzameling zijn van meerdere (informatie)systemen die allen nodig zijn om die functionaliteit te kunnen leveren. De volgende vraag moet hier beantwoord kunnen worden - "Wat kan ik als eindgebruiker met .... zodat ik informatie beschikbaar kan stellen of zelf kan beschikken over juiste informatie"
01. Metamodel
ondersteunen System Use Cases (Zorg)activiteiten
realiseren Diensten System Use Cases
realiseren informatiesystemen(en) System Use Cases
