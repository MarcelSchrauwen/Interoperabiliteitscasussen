Digitale zorg [Koppeltaal] ()
Digitale zorg zoals dat via op Koppeltaal aangesloten applicaties ondersteund kan worden. Use cases: https://vzvz.atlassian.net/wiki/spaces/KTSA/pages/27143996/Het+kiezen+toewijzen+uitvoeren+en+volgen+van+interventies Mogelijk scenario: Een zorgverlener en patiënt bespreken tijdens een afspraak in de zorginstelling een mogelijk behandeltraject. Het zal een hybride vorm worden waarbij de reguliere zorg gecombineerd gaat worden met digitale zorg.
