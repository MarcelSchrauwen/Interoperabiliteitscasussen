Aanvullend onderzoek [msz] ()
Uitvoering van aanvullend onderzoek zoals klinisch chemisch onderzoek, röntgen onderzoek, microbiologisch onderzoek, pathologisch onderzoek, etc. In feite is er een domein onderzoek met daarin specialisaties naar het type onderzoek. Binnen het RDZ kunnen voor specifieke toepassingen specialisaties toegevoegd worden.
