Zorgaanbieder specifieke Koppeltaal-voorzieningen ()
De koppeltaal-voorziening is opgedeeld in vier componenten waarvan er twee zorgaanbieder specifiek zijn. Iedere zorgaanbieder beschikt over een eigen implementatie van deze componenten. Zorgaanbieders of leveranciers kunnen er voor kiezen om deze componenten zelf te bouwen en te leveren, bijvoorbeeld vanuit het eigen datacenter.
Zorgaanbieder specifieke Koppeltaal-voorzieningen Autorisatie server [Koppeltaal]
Zorgaanbieder specifieke Koppeltaal-voorzieningen FHIR store [Koppeltaal]
