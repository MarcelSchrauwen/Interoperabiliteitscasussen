Koppeltaal FHIR service ()
De gehele reis door een digitale zorg interventie wordt ondersteund door slimme samenwerking tussen de op Koppeltaal aangesloten applicaties. Door deze 'integratie' kan de zorgverlener en cliënt zonder admnistratieve obstakels en overbodige handelingen optimaal samenwerken aan de behandeling van de cliënt via digitale zorg. Deze integratie wordt mede mogelijk gemaakt door het delen van data door de op Koppeltaal aangesloten applicaties.
