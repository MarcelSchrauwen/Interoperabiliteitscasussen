Voorziening-beheerder identificatie [Koppeltaal] ()
Het overleggen van een geldig identiteitsbewijs waarmee de identiteit van een voorziening-beheerder kan worden vastgesteld. Voor de Koppeltaal-voorziening leverancier gelden specifieke eisen met betrekking tot identificatie van voorziening-beheerders. Deze eisen zijn vastgelegd in een overeenkomst tussen VZVZ en de voorziening-leverancier.
