CA.GZ.001 - Gezamenlijke zorg rondom een VG cliënt ()
Medische zorgprofessionals verbonden aan een samenwerkingsverband in de regio die ‘medebehandelaar’ worden van een VG-cliënt waar ze geen dossierhouder van zijn, kunnen de actuele en noodzakelijke medische gegevens van deze cliënt niet raadplegen.
