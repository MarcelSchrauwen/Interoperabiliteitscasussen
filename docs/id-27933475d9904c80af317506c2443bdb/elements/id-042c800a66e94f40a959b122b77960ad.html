(Zorg)activiteiten ()
De System Use Case beschrijft het gedrag (de functionaliteit) van een persoon of personen (via een rol). De nadruk ligt daarbij vooral op wat gedaan moet worden om het (zorg)proces optimaal te laten verlopen en welke informatie daarbij nodig is. Een business use case wordt idealiter eerst volledig los van techniek (de system use case) uitgewerkt. De system use case is het gevolg van een behoefte vanuit een business use case. Binnen de referentie domeinmodellen worden business use cases o.a. bedrijfsprocessen, werkprocessen, en activiteiten genoemd.
