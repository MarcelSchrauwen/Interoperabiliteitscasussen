LaboratoriumUitslag ()
Een laboratoriumuitslag beschrijft het resultaat van een laboratoriumbepaling. Het gaat hierbij om monster-geörienteerde bepalingen zoals deze verricht worden in laboratoria als Klinische chemie, Serologie, Microbiologie etc. Behalve de uitkomsten van testen met een enkelvoudig resultaat kan dit concept ook de uitkomsten van meer complexe testen met meervoudige resultaten of 'panel' bevatten. bron en meer info: https://zibs.nl/wiki/LaboratoriumUitslag-v4.6(2020NL)
