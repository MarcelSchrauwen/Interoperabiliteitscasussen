Beschikbaar stellen patiëntinformatie [msz] ()
Het op digitaal of op papier, verstrekken of versturen van medische informatie over een patiënt aan externe zorgverleners. Uitwisselen kan ook beschikbaar stellen kan zijn en e.g. via notificatie. Verslag is zoals het nu gaat, maar zou ook beschikbaarstellen kunnen zijn e.g. via bijvoorbeeld XDS / MedEx artsenportaal. Zie ook "IO Uitwisselingsinformatie".
